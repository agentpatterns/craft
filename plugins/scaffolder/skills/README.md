# Scaffolder Skills

Skills for software architecture patterns.

| Skill | Command | Description |
|-------|---------|-------------|
| [hexagonal-architecture](hexagonal-architecture/) | `/hexagonal-architecture` | Applies hexagonal (ports & adapters) architecture with domain-first design |
| [adr](adr/) | `/adr` | Guides writing minimal Architecture Decision Records with advice-process awareness |
| [scaffold-ts](scaffold-ts/) | `/scaffold-ts` | Scaffolds DDD TypeScript projects from Gherkin feature files |
| [likec4-c4](likec4-c4/) | `/likec4-c4` | Generates structural C4 diagrams using LikeC4 DSL |
| [likec4-dynamic](likec4-dynamic/) | `/likec4-dynamic` | Generates workload flow diagrams using LikeC4 dynamic views |
| [data-flow](data-flow/) | `/data-flow` | Creates Data Flow Diagrams using Mermaid notation |
